# 退休工资计算APP（H5版）

一个用于计算退休工资和查看退休倒计时的移动端Web应用。

## 功能特性

### 1. 首页 - 退休倒计时
- 根据用户信息计算离退休还有多天
- 直观的倒计时展示
- 显示退休日期和剩余天数

### 2. 退休金计算
- 根据用户提供的信息计算退休时可以领到多少退休金
- 支持输入：工龄、基本工资、社保缴纳情况等
- 提供多种退休金计算方案
- 显示详细计算过程

### 3. 退休资讯
- 最新的退休政策资讯
- 退休相关的新闻动态
- 退休生活指南
- 智能分类推荐

### 4. 我的 - 用户信息管理
- 填写和管理用户信息
- 保存计算结果
- 个人设置
- 历史记录查看

## 注册功能
- 支持手机号码注册
- 支持邮箱注册
- 用户信息加密存储

## 技术栈
- 前端：HTML5 + CSS3 + JavaScript（原生）
- 响应式设计，支持手机端浏览
- 本地数据存储（LocalStorage）

## 使用方法
1. 在浏览器中打开 `index.html`
2. 根据需要填写用户信息
3. 进行退休金计算或查看退休倒计时
